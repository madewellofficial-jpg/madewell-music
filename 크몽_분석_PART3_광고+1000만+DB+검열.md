# 9단계. 월 매출 1000만 원 프로젝트 연결

## Q1. 크몽에서 월 1000만 원 구조의 일부를 만들 수 있는가?

**일부는 가능, 전부는 불가능.** 크몽 단독으로 월 1000만 원을 만들려면 믹싱 기준 월 40~50건, 레슨 기준 월 33~40회 이상이 필요하다. 리뷰 0 신규 판매자에게 그건 현실적이지 않다. 단, 크몽의 역할은 다음이다:

- **월 100~200만 원 기여 (현실적 목표)**
- **첫 리뷰·첫 신뢰 데이터 확보**
- **어떤 고객이, 어떤 언어로, 어떤 서비스를 사는지 실제 데이터 수집**
- **메타 광고·블로그·랜딩페이지로 확장할 때 '이미 검증된 상품'으로 쓰기**

크몽은 1000만 원 프로젝트의 **테스트 마켓이자 신뢰 베이스**다. 전체 매출 구조의 핵심이 아니라 진입 채널이다.

## Q2. 어떤 상품이 크몽에서 가장 가능성이 높은가?

**믹싱·마스터링 BASIC 7만 원 상품.** 이유:
- 파일 기반이라 심사 통과 가능성 최고
- 구매 결정이 빠름 (파일 보내면 됨)
- 리뷰 확보가 쉬움 (납품 후 즉시 요청 가능)
- 반복 구매 가능 (같은 고객이 다른 곡으로 재주문)

## Q3. 믹싱·마스터링, 녹음, 레슨 중 메타 광고와 가장 잘 연결되는 것?

**레슨 > 녹음 > 믹싱 순서.**
레슨은 전·후 비교(보컬 교정, 작곡 완성)가 영상으로 가장 잘 보임. 광고 소재 만들기도 쉽고, 단가가 높고, 반복 구매(4회→8회)도 된다.

## Q4. 크몽에서 첫 리뷰를 어떤 상품으로 확보해야 하는가?

**믹싱·마스터링 BASIC 7만 원.** 조건:
- 구매자가 이미 파일을 갖고 있음 → 진입 장벽 없음
- 파일 받고 만족하면 바로 리뷰 남길 수 있음
- 가격 부담이 낮아 "한번 써볼까" 결정이 쉬움

## Q5. 리뷰가 쌓인 뒤 어떤 상품을 가격 인상하거나 분리해야 하는가?

- 리뷰 5개 이상 → 믹싱 BASIC 7만 → 9만 원으로 인상
- 리뷰 10개 이상 → 보컬 레슨을 미디 레슨과 분리 등록 (이때 이미 레슨 리뷰도 생기면)
- 리뷰 20개 이상 → PREMIUM을 분리해 "발매급 믹싱 패키지"로 고가 포지션 구축

## Q6. 저가 경쟁 피하면서 중상가 포지션으로 가려면?

**3가지가 동시에 필요하다:**
1. **썸네일:** 스튜디오 장비 실사진 → 즉각 시각 차별
2. **제목:** "파일 → 음원처럼"의 결과 중심 언어
3. **가격:** BASIC은 낮게(7만), STANDARD는 중간(15만), PREMIUM은 가치가 보이게(25만)

이 세 가지가 동시에 맞으면 "저렴하진 않은데 믿을 수 있겠다"는 인식이 생긴다.

## Q7. 이번 달 해야 할 실행 순서

1. **지금 바로:** 믹싱·마스터링 등록문 크몽 재제출 (PART 2 복붙)
2. **1~3일 내:** 보컬·랩 녹음 등록문 동시 준비 완료
3. **1~3일 내:** 1:1 음악 레슨 등록문 동시 준비 완료
4. **믹싱 승인 확인 후 3~5일:** 녹음 서비스 제출
5. **녹음 승인 확인 후 3~5일:** 레슨 서비스 제출
6. **첫 구매 발생 후:** 납품 후 구매자에게 리뷰 요청 메시지 보내기
7. **리뷰 5개 확보 후:** 가격 조정 + 메타 광고 테스트 준비

## Q8. 크몽에서 팔리는 상품과 메타 광고로 팔릴 상품은 같은가, 다른가?

**다르다.** 크몽 → 파일 처리·레슨 서비스 (저~중가, 명확한 구조). 메타 광고 → 레슨(결과 변화 보여주기), 프리오더, 브랜드 경험이 더 잘 작동한다. 단, 크몽에서 검증된 고객 언어(불안·욕구·리뷰 키워드)를 광고 카피로 재사용하는 것은 가능하고 효과적이다.

## Q9. 크몽은 장기 핵심 채널인가, 테스트 마켓인가, 리뷰/신뢰 확보용인가?

**현재는 테스트 마켓 + 신뢰 확보용.** 장기적으로는 크몽이 자동 유입 채널이 될 수 있지만, 1000만 원 구조의 핵심은 아니다. 크몽은 "이 사람은 검증됐다"는 리뷰 베이스를 만들고, 그 리뷰를 랜딩페이지·블로그·메타 광고에서 소셜 프루프로 활용할 때 진짜 가치가 나온다.

## Q10. 추가 데이터와 결합하면 어떤 전략이 가능한가?

추가 데이터(현금흐름, 메타광고, 프리오더, 팬덤, 릴스, 블로그, 브랜드 톤)와 결합하면:

- **크몽 리뷰** → 랜딩페이지 소셜 프루프로 사용
- **크몽 고객 언어** → 메타 광고 카피에 재사용
- **크몽 상품 구조** → 프리오더/레슨 패키지 가격 기준 설계
- **크몽 판매 데이터** → "어떤 서비스가 실제로 팔리는지" 근거
- **크몽+블로그 SEO** → 믹싱/마스터링/레슨 키워드로 무료 유입
- **크몽+릴스** → 작업 과정 공개로 자연 유입 → 크몽 검색 증가

전체 구조: 릴스·블로그 유입 → 크몽/랜딩페이지 → 레슨/믹싱 → 리뷰 축적 → 메타 광고 확장 → 프리오더/고가 패키지

## 경쟁자 제목 패턴
- [서비스명]+[해드립니다] — 가장 많음 (검색형)
- [감각적 결과]+[서비스명] — 소수지만 차별적 (M01)
- [경력/권위]+[서비스명] — 증빙 필요
- [속도]+[문제해결] — M03 (긴급 수요형)
- [DAW명]+[레슨종류] — L01, L02 (레슨 카테고리)

## 경쟁자 썸네일 패턴
- 텍스트+단순배경 — 가장 많음 (저가형)
- DAW 화면+텍스트 — 중간급
- 장비사진+텍스트 — 고급형 (희소)
- YouTube 영상형 — M06 (차별적이나 제작 필요)

## 경쟁자 가격 패턴
- 저가: 2~4만 원 시작, 대량 거래
- 중간: 5~10만 원 시작, 소량 거래
- 중상가: 8~15만 원 시작, 리뷰 100개+ 가능

## 경쟁자 약점
- 대부분 썸네일 퀄리티 낮음
- 저가 강자들은 무제한 수정으로 심사 리스크
- 경력 표현으로 증빙 없이 무리하는 케이스 다수
- 레슨 상품들이 오프라인 주소 직접 노출
- 파일 준비물·작업 범위 불명확한 상품이 많음

## MADEWELL 차별화 포인트
1. 실제 스튜디오 장비 사진 → 썸네일 즉각 차별
2. 녹음+믹싱 연결 포지션 → 시장 내 독자 포지션
3. 보컬 레슨+녹음 피드백 결합 → 레슨 시장 내 차별
4. 심사 통과 가능한 문구 구조 → 경쟁자들이 반복 걸리는 부분 회피
