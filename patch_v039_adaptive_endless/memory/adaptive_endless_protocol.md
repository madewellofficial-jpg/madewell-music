# MADEWELL Adaptive Endless Protocol v0.1

끝장토론은 무한 반복이 아니라 “조건부 연장”으로 운영한다.

## 핵심 원칙
- 한 번씩 말하고 끝나는 발표회는 부족하다.
- 그러나 무한 재반박은 토큰 낭비와 말싸움 루프가 된다.
- 정답은 “최대 추가 라운드 + 계속할 조건 + 중단 조건”이다.

## 계속할 조건
아래 중 하나가 새롭게 발견되면 추가 재공방을 허용한다.
- 새로운 실패 가능성
- 후보 점수 변화
- 새 폐기/수정 후보
- Red Team의 치명 리스크
- 남은 쟁점이 실행 전 반드시 줄어야 하는 경우

## 멈출 조건
아래에 해당하면 토론을 멈추고 실행으로 넘어간다.
- 같은 말 반복
- 상위 후보가 명확함
- 남은 쟁점은 토론이 아니라 실험으로만 확인 가능
- 추가 토론보다 실행 데이터의 정보값이 큼
- 오늘 실행안이 충분히 압축됨

## 최종 목표
오래 떠드는 것이 아니라, 더 토론해야 할 때와 지금 실행해야 할 때를 구분한다.
